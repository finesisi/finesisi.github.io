《病毒星球》读后感——从微观了解疫情载体

发表于 2020-02-23 | 更新于 2020-02-25 | 分类于 读书笔记 | | 次阅读

《病毒星球》作者 卡尔·齐默（Carl Zimmer），知名科普作家。他在耶鲁大学教授科学和环境写作，该书是其普及病毒知识的一本书。

如果不是疫情，我怕是很难去接触这样一本科普作品。
面对疾病，人类并不会恐惧，直到它发生在我们身上，与我们工作、生活、家人息息相关。

为什么读

2020年春节，新冠病毒肆虐，从武汉扩展到全国。每天成千的确诊病例，对应的是每天着数以百计的死亡数字。研究表明，死亡数字并不会引发太多情绪，因为不够“鲜活”。更何况每天不断增加的数字，容易让人变得习惯和麻木。
然而每个数字背后其实都是一个个鲜活的生命，影响和关联着由这些生命组成的家庭。他们是导演、是院士、是教授、是医生，他们也可能是你和我。
疾病面前人人平等。了解致病源头，至少可以更好的对它引起重视。

微观世界里的病毒

从《病毒星球》中，你会知道：
1、这个世界充满了病毒。在在处处皆是病毒。从植物到野生动物、从野生动物到人类。各种生物与病毒共处在这个星球中。
2、特定的病毒感染只感染特定的动物、植物。例如每升海水里含有1000亿个病毒颗粒，其中不同的病毒可能感染某种海藻或者鱼类。又比如HPV喜欢感染女性的宫颈。
3、人类将永远和病毒“和谐共处”。一方面病毒能协助我们产生氧气，让我们得以呼吸；另一方面，病毒时不时侵蚀人类个体，有助于个体免疫力提升。
4、在中国，多数人的大型病毒流行事件记忆仍然是SARS（2003年），以及现在的新冠。事实上这一二十年世界的很多地方也爆发了其他病毒流行事件。从全球看，艾滋病已耳熟能详。但毕竟更多通过性或血液传播，普通人尚且能用相关措施防御。而其他一些病毒引发的流行病就没有那么幸运了。亚洲、美国多次爆发禽流感；2014年非洲、美国，埃博拉病毒流行，一些死者七窍流血而死，死状惨烈，从发病到死亡速度极快。据说非洲蝙蝠可能是传播源；2012年，沙特阿拉伯的MERS病毒流行，骆驼、非洲蝙蝠是其传播源。
5、在你有生之年，下一场瘟疫还会来临，这绝非危言耸听。地球就像一个容器，温度的变化，会导致整个星球环境温湿度产生巨大变化，从而原本平衡的病毒群变得不安分起来。逐步变异、演变，最终可能变成巨大的灾难。

切断人类与病毒的关联和因果

一方面人类总是在刻意制造“关联”。例如吃野生动物，侵蚀野生动物栖息地，将原本只会在动物之间传播的疾病，引火到了自己身上。例如艾滋病起源于大猩猩。人们捕杀大猩猩、甚至吃他们的肉，导致人类致病。禽流感起源于鸟类，而鸟的消化道和人的呼吸道细胞表皮特别像，非常利于这些病毒繁殖，人通过吃鸟类，间接感染上了禽流感；另外还有马，三分之二的健康马都有乳头瘤病毒，也可以传染给人类。

另一方面，人类在无意制造“因果”。全球气温变暖，导致病毒生存环境发生巨大变化，甚至使得远古的一些病毒重新苏醒。早些时候，人类战胜了天花、脊髓灰质炎。从对病毒的恐惧到战胜病毒，赢得胜利。这个背后，有无数因病死亡的人作为代价。人类以为自己能一次又一次战胜疾病。事实上，不是我们战胜了疾病，而是疾病放过了我们。曾经的霍乱最终自然消亡，曾经的SARS逐渐平息，很多都是因为天气变化、自然界其他因素导致病毒无法生存，而逐渐消亡。并非人类发明了疫苗，让病毒无法侵蚀人类。
人类在不停的造因，如何不会承受其果呢？

查理·芒格说了一句更精准的话：“宏观是我们必须接受的，微观才是我们可以有所作为的。”

看完《病毒星球》后，又追溯看了《血疫》，然后又看了更微观的《从一到无穷大》，从微观世界到宏观世界，从病毒到宇宙，人类历史就一直在与疾病斗争。天地以万物为刍狗，人类不过是生命的一环，因此面对自然仍需敬畏。
生命其实非常脆弱。我们现在活着，是我们的祖先，绕过各种危险概率事件（战争、瘟疫、飞来横祸、地震、火灾、谋杀等等），侥幸得以存活。一代又一代，到了我们这里。生命本不易，从微观上看，至少对于个体而言，我们可以做到不吃野生动物；我们可以做好各种预防措施；我们可以进行低碳生活。
