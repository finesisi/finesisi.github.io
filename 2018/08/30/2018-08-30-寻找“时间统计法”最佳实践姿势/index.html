寻找“时间统计法”最佳实践姿势

发表于 2018-08-30 | 更新于 2018-11-18 | 分类于 效率工具 | | 次阅读

时间令人烦恼之处在于它不能不用。

柳比歇夫奇特的一生

《奇特的一生》讲的是柳比歇夫奇特的一生。为何奇特？其人著作等身，精力旺盛，博学多识，成绩斐然。这些成就赫然看起来高不可攀，然而他所使用的法宝却是一种普通得不能再普通的方法，记录时间。我们将这种方法称之为柳比歇夫的“时间统计法”。为何说它普通？因为它简单易操作，普通人一样可学。难的地方在于坚持。毕竟天才不是一日成就的。

《奇特的一生》并非柳比歇夫所作，是后人返遡研究他的一生所写。时间统计法上手简单，难点在于实践，更难的地方在于坚持。个人感觉书中内容表述累赘，如要学习了解其方法即可。如特别有兴趣也不妨一读。

带给你什么

  1. 通过时间统计法，唤起自己对时间的敬畏。
  2. 通过时间统计法，可以使自己生活更高效，可以一步一步找到自己能力极限。
  3. 时间统计法是一种超越自身可能性的方法，它可以节约时间，但不能额外增加时间。它的关键不在时间的数量上，而在时间的质量上。
  4. 有了孩子之后，时间统计法可以记录你与孩子相处的点滴，从美好的画面到画卷铺展开来。你的这种习惯在积累与沉淀中将让孩子敬畏，你将成为孩子的学习榜样。

怎么做

  1. 时间统计法记录时间：格式为：日期+事件+花费时间
  2. 将一昼夜纯时间分为三大类：中心工作、非中心工作、需要创造的任务。每项活动占据的时间需与重要性相称。
  3. 每天坚持记录时间。
  4. 月底/年底总结时间使用情况，写每月／每年小结，看了什么书，做了多少娱乐，分别是哪些，直观多少。
  5. 生活上让一切都井井有条。例如所有自己的文字都分类编号，自己便于查找。电脑上的文件一样。

tips：

  1. 要达成人生目标需要耗费巨大的时间，所以应不断从生活中挖掘时间。
  2. 什么是纯时间？刨除掉任何间歇的时间，只花费在你这项工作上的时间。柳比歇夫做过测算，一昼夜的纯时间为10个小时。
  3. 制定自己的时间使用守则，保证身体健康。例如：不接受紧急任务；累了就休息；每天睡觉10小时（柳比歇夫活了80多岁，和他注重休息，张弛有度的生活密不可分）。
  4. 严格筛选自己的日程活动，例如避免串门，庆祝活动等。

我的实践

实践第3个月

践行柳比歇夫的时间统计法3个月，总体感受效果有限。直到近两天悟出一个道理，其实我不应记录自己做了什么，而应记录我输出了什么。这两种方式唤起的意识完全不同，前者唤起时间分配意识，后者唤起要事为先、以终为始等各种意识。当然，时间统计法也有个好处，当你回顾自己做了什么的时候，你看到这些文字记录的事件，会觉得生活很美好；而如果仅记录输出，虽然成就感十足，但也会遗忘很多生活点滴吧。世上难得两全法，继续实践中。
（写于2018-02-01）

实践第8个月

从原汁原味记录时间（实践3个月）–只记录重要输出（实践1个月）–回归记录所有时间（实践4个月）–决定只记录重要输出（从现在开始）

这其中主要因素是习惯的改变。
为什么我又重新回到只记录重要输出？

  1. 记录原汁原味时间比较花费内省时间，有时候会感觉比较烦；
  2. 自身日行惯例的时间已经固定，例如吃饭、睡觉、洗澡、洗漱等，基本时间固定，日后用来内省这些时间意义有限；
  3. 未来行动：今后记录时间主要集中在：
  • 工作内容
  • 创造
  • 温故而知新
  • 高质量陪伴
  • 购物等闲暇
    （写于2018-07-05）

实践第8.5个月

过了10天只记录重点的“休闲”日子，仅仅新鲜了头几天，后面既记录没了动力，连做事情也没了动力。凡事相辅相成，长短相和，声音相随，如果仅记录增值，就凸显不了不增值的地方。如果在频率强化过程中，仅全是正强化，那也就不存在正强化了。正因为此，更加深感时间统计法中，不仅记录“中心时间”，同时记录“非中心时间”的意义。

由此，以上7月5日的感受证明是虚幻的呈现，还需要有耐性地按照柳比歇夫时间统计法要求严格执行，这样才能最大收获。过去7个月实践表明，真实不虚。
（写于2018-07-16）

实践第10个月

以前用过用软件iTimeLog统计时间，这次改用纸笔记录，感觉更容易引发思考。正如花钱使用纸币还是扫支付宝／微信一样，软件记录和纸笔记录可能激活的大脑区域也不太一样；
（写于2018-08-11）

实践第10.5个月

终于找到实践最佳姿势，即完全回归纸和笔记录时间。白天如实记录。晚上录入电脑归档时间使用情况。仍然坚持周总结、月总结、年度总结。
（写于2018-08-30）
