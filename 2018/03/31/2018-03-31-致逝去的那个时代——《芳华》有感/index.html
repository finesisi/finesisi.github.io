致逝去的那个时代——《芳华》有感

发表于 2018-03-31 | 更新于 2018-11-18 | 分类于 电影感受 | | 次阅读

《芳华》有模仿《阿甘正传》的痕迹，试图通过一组组小故事串联出时代的变迁。《阿甘正传》里，女主出现篇幅不多，只是每次出现，观众自动脑补她的经历。而《芳华》，它很贪心。同时写了男主和女主两条线，试图在某个时候，通过一种冥冥之中的命运将他们交织在一起。想试图通过两条线，两个小人物，展示时代变迁，这绝非易事。

多年后，我一定记不住里面的男主角。先撇开他的演技不说，人物性格不够鲜明。一个老好人，就像身边的大树，这种遮风挡雨的安全感似乎变得理所当然，太过沉默。没有智慧、烂掉的老好人，终究会被自己的性格所害。
这是一部灰色基调的电影，不悲不喜。历史如同洪流滚滚向前，小人物在里面挣扎几下，然后被无情淹没。它没有按常规思维模式出牌。要是好莱坞的模式，何小萍，那个遭人嫌弃的姑娘，应该在努力勤学苦练后，逆袭成为一名大舞蹈家；刘峰，那个曾经的老好人，应该在被人诬陷后，黑化成一个不择手段、手握大权的领导；穗子，那个纯粹而美好的姑娘，应该被揉碎了才是，这样印象才够深刻；至于那个丁丁姑娘，应该拒绝刘峰、远嫁海外后，过得很不好，后悔当年没有选择刘峰了才是。可是这部片子，偏偏自然而然地发展，没有刻意制造这些冲突，也正因为如此，让人感觉真实。
小人物内心小宇宙终究只能被现实熄灭，那种无力感冲不出时代，只留下残忍的烙印。何小萍，从小没有父母关怀，受尽欺凌，即便进入了新环境，没有一个早期铸就的强大内心，凭什么逆袭成为一代芳华？刘峰，一个木匠的孩子，凭什么要求他有强大的野心，大胆追求心中所爱，成为人生赢家？至于那个昆明军区司令的儿子陈灿，并没有因文工团垮掉，就没法过得滋润，他凭借父亲的资源，照样能过得风生水起，有声有色。那个后来成为陈灿老婆的姑娘，凭什么敢大胆追求陈灿，因为她觉得自己跟他倒也门当户对。也正是这句话，让一直深爱陈灿的穗子泄了气，索性撕掉了她写给陈的那封情书，只能将感情吞到肚里，默默垂泪。这部片子看到的是小人物在一个时代的众生相，很真实，不戏剧，很自然。
印象最深的是开篇穗子在镜头前的那段舞蹈，年轻姑娘脸上的笑容，悠扬的交响音乐，阳光打在他们身上。那才是真正的芳华。之后的每一个故事，都是在破坏这个美好的画面。在旁边看着穗子跳舞、满怀对未来憧憬的小萍，后来却偷拿了另一个姑娘的军装去拍照；舞台上看起来纯粹无邪的姑娘们，却对可怜的小萍各种嫌弃和排挤；老好人刘峰被诬陷下放到偏远连队，最后在前线上失去右手；遭人嫌弃的小萍下放到前线做护士，目睹太多血腥场面，最后逼疯得了精神病；连那张背景–文工团也没了，曲终人散。当刘峰再次回到空荡荡的文工团排练室的时候，他仿佛看到了这样一个画面。当年阳光洒进来，几个姑娘用那优美的姿势舞动着，整齐地在排练的画面。当刘峰走过去的时候，又回到了现实，教室那样阴暗，仿佛阳光被黑洞抽去，寒气逼人。那一切恍如隔世。曾经的阳光少年也不复存在了，仿佛在诉说另一个人的故事。
