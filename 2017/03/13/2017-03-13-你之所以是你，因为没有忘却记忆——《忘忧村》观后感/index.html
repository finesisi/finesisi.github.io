你之所以是你，因为没有忘却记忆——《忘忧村》观后感

发表于 2017-03-13 | 更新于 2018-11-18 | 分类于 电影感受 | | 次阅读

“忘忧村”三个字，脑中浮现《桃花源记》里的世外桃源，犬牙交错，鸡犬相闻，不知有魏晋；或是“柳暗花明又一村”，或是像武林群侠传中琴棋书画酒茶花医各类隐士聚集在山中，谈天谈地，好不逍遥……

电影中描绘了另一个桃花源，它并不美好。这里集聚着人的各种欲望。当初那些最可怜、你最同情的人，他们在掌握权力后，反而表现更为残暴。

所有的罪源源于一个周朝的“忘忧”神器。一个假道士利用这个真神器有意识的【裁剪】大家的记忆，把他们变成了另一种人，还贴上标签，所谓“不重要”的人、“不痛不痒”的人、甲乙丙丁等各种毫无特色的人等等。你可以变出你想要的任何人。换句话说，你不再是你，他不再是他。

它将人复杂性格刻意简化了。然而，人的性格岂是独由记忆决定的？单纯的抹掉记忆就可以塑造出想要的人吗？即使是婴儿也是自带气质和技能的啊，作为成人一旦通过“忘忧”就径直忘掉了全然的自我，包括外在的身份以及天生的性格。显然不符合认知科学。

不过这种简化的方法倒真可以作为一种工具用在我们自己身上。例如选择性记忆和选择性遗忘某些影响自己的关键事件，刻意忘掉一些那些造成自己阴影的关键事件等等。除了忘记还有另外一种方法就是弱化。如何弱化？先强化之再弱化之。如果一个事务在脑中挥之不去不如将情绪大胆发泄，发泄完后一切回归，回到也无风雨也无晴的状态。就像在白天那不经意的一瞬间某种意向进入了潜意识，那一霎那让自己有所意识后，再运用自己的大脑特意将这种瞬间的意识进行强化，晚上便不会化作自己的梦境。又如因某些事情触发出某种情绪，若当时强烈，未来同样事物再难触发出同样强烈的情绪，这种情绪会比之前的情绪弱很多。这种情况在恋爱中尤其明显。而这里用到的方法仅仅是【删除】，显然简化了控制记忆的方法。

最后，借助《清醒思考的艺术》中提到的一个理论，人总是要给理由的，给出理由就会相信。上映版的《忘忧村》实际上是个阉割作品。结局中女主人公秋蓉说自己已将记忆都还给了村民。但我们看到的情况是，村民将秋蓉推为村长，一副毛主席万岁的样子，一排排甲乙丙丁、不重要的人、不痛不痒的人，一如回到了道士统治村子的时代。这像是还回记忆的样子吗？显然不符合逻辑！查询第二结局后才知道，原来秋蓉将村民记忆全部烧毁，这就能解释通了，这样的理由和现象才全部是符合逻辑的。然而看第一遍的时候我几乎也相信了。果然是给理由我就信啊。自己都陷入到这样的圈套去了。

电影本身一般，但是一部两小时的电影竟然给了我这么多反思，还是很值得的。
