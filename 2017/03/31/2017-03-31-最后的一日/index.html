最后的一日

发表于 2017-03-31 | 更新于 2018-11-16 | 分类于 个人创作 | | 次阅读

1860年9月21日，法兰克福

8:00

如果说最近有什么不适的地方，昨天倒是有一些。

昨日8:00

我感觉身体一阵抽搐，然后不可控地跌倒在地，碰上了额头。额头很快肿起来。按医生的建议，我最好记下这些与身体状况有关的信息。

管家进书房的时候，我刚洗完冷水澡，正喝着咖啡。他打开窗子，清新的空气随着拉开窗帘的“嘶嘶”声袭进房间，温和的阳光散落在书房中那尊镀了厚厚金箔的佛像上，而康德、歌德、笛卡尔的画像则依旧在书房角落静静思考，爱犬阿托姆则伏在我身旁，它正盯着那各式各样的狗雕像发呆。

管家问，先生，您的额头怎么了？需要叫医生过来吗？我嘴上说不需要，心理却想着，可不能让纳西小姐看到我现在的样子。

管家走后，我开始独享属于自己的时光。我以往的作品几乎都在这里完成。一个作品要想名副其实，只可能是自由人完成。我应该庆幸自己是个自由人，而不是干粗活女工的孩子。虽然所谓的名声来得姗姗来迟，但我已不在乎。毕竟，使命已经完成。虽然多年前已封笔，但这并不妨碍我每天继续思考。我享受其中，终生如此。

昨日11:30

我穿上燕尾服、打上洁白的领结，去饭店吃午饭。我只去“英国饭店”，几十年如一日。我并非对这家饭店情有独钟，只是因为这样省时间。我不用花额外时间思考原本很简单的事情。值得一提的是，那家饭店的傻瓜显然比其他地方少。

虽然就餐过程中仍然充斥着同一样性质的社交谈话，门口依旧站着那么多四处张望内心空虚无比的人。但是这一切都不重要。曾经，这种观察入微到感觉对我来说太过于强烈，细微的骚扰都会让人极度不耐烦。然而这么多年过去了，这反而让我内心更加平静。我终生都在追寻着自己的个性与外界的一致——虽然这种平衡很难达到。但是我已经学会与他们相处。他们无聊的举动倒成了思考的素材。而现在，我只是玩味和摸索着内在世界和外部世界的各种现象，甚至又有了将它们组合的冲动。这个过程妙不可言。

昨日13:00-16:00

虽然阅读无法代替思考，但是下午的时间并不适合创作。只有苏格拉底、亚里士多德那些经典值得耗费此刻的时间。刚刚，那黑格尔骗子的言论竟在脑中闪过，我花了片刻才将他从脑中抹去。

昨日16:00

与阿托姆散步。每天2小时的快速运动对于维持身体健康至关重要。

昨日18:00

图书馆阅读《泰晤士报》，正如父亲说的，“从《泰晤士日报》，你可以知道你想要知道的一切。”看看报纸，就知道人性始终未变。正如伏尔泰说的，“当我们离开这个世界点时候，这个世界还是照样愚蠢和邪恶，跟我们刚来到这个世界的时候所发现的并没有两样”。

昨日19:00

依旧去英国饭店享用晚餐。很可惜，那枚金币还没有送出。

昨日21:00

阅读《吠陀》，从它的一页纸著作，我们所学到的就超过康德以后的整卷整卷的哲学著作。阅读《吠陀》，仿佛清冽的泉水从头上倾泻下来，带着这种感觉入梦同样妙不可言。

9:00

清晨的时光依旧那样明亮、清新和轻松。此时此刻，我仿佛听到一种声音，它告诉我，值得怀念却又无法怀念的一天即将来临。写到此处，我头开始发胀，眼前出现重叠的影子。

我即将孤身上路了。谁要是像我那样孤独地过一辈子，自然更懂得那种孤身上路是怎么一回事。

我穿过窗子，掠过草地、篱笆、石子小路，我看到康德、笛卡尔、歌德，他们手拿月桂花环，向我微微颔首。花环被柔和的阳光萦绕，散发出淡淡芳香。

原来，我并非孤身上路，无论生死，真理和益友一直如影随行。

后记：

我模仿叔本华的口吻，以日志的方式记录他临死前一天的生活。对于这个无与伦比的哲学家来说，几十年如一日的规律生活只是框架，追求真理的过程才是趣味。能够在死的时候看到真理和益友来接引他，死的那一刻发现自己并非孤身上路，这也许是叔本华最为欣慰的事情吧。
