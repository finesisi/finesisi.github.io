《儿童的人格形成及其培养》读后感

发表于 2017-11-11 | 更新于 2018-11-18 | 分类于 读书笔记 | | 次阅读

作为个体心理学开山鼻祖，源头性的“毒物”，简直深深“毒害”我心。侃侃而谈且质朴坚定的语言，仿佛感觉自己正在阅读叔本华的著作。这本书由韦启昌先生翻译，韦先生也翻译过叔本华多本著作，想想先生品味一定不错，于是按图索骥到此书。阅读过程中，不断产生“原来如此”的感觉，多次唤起自己童年的记忆。成人后的一些习惯原来可在童年经历中找到对应点。其著名观点“自卑情结”，能作为源头解释多种现象。

孩子真的表现得好／坏吗？你看到的只是表象！

人格重不重要？非常重要，它就像一个滤镜。当信息投射时，面对同样的信息不同人有不同反应。所以人们所看到的东西，往往是经过了人的整体人格的重整后反映出来的。阿德勒提出，“培养孩子健全人格——这才是儿童教育的首要目的。”本书专注孩子人格培养，解药就是防止自卑，同时指出生活中可能导致孩子自卑发生的多个雷区。为此，作者还提供多种建议。简直就是苦口婆心教你怎么育儿嘛。

什么样的孩子才是人格健全的孩子呢？书中给出了判断标准，我深表认同：独立、自信、勇敢、不畏困难，以及具备与他人合作的意识和能力。

当我们看到孩子的一些行为时，不要妄下定论。行为多种多样，孩子行为背后的原因更为重要。不要死咬着表象不放。如果我们直接问孩子行为背后的原因，这非常难，甚至连孩子本人都不一定清楚原因，你又怎么可能问得出来？

那么，如何找到孩子行为背后的真正原因呢？应该研究他连续的行为，而不是割裂的片段。一个富有经验的心理学工作者应该能根据一些行为大致不差地塑造出人格的整体印象。这个印象塑造出来之后，还能预测此人的一些行为。这才算是了解了他的人格特质。

哪些行为可作为线索判断？
● 孩子的眼睛，是否敢正视大人？
● 孩子是否依靠什么站着？
● 与大人谈话的距离？
● 孩子走路的姿势？
● 孩子在学校的社交情况？
● 孩子对老师的态度？
● 观察孩子喜欢什么类型的读物？
● 孩子的梦境？
● 将孩子放到意想不到的新环境中，看他怎样表现？

书中提到，极端的两极判断只是孩提时代思维的体现。我们在判断孩子行为时，如果认为一些孩子的行为是“正确”的（阿德勒认为对于社会有益的事情可以视为“正确”）或是“不正确”的，也许就陷入了这种幼稚的思维当中。事实上，孩子行为表现正确，也许只是暂时的，如果目标和动机存在问题，发展下去，可能成为问题少年。那些表现出来各种“不正确”行为的孩子，也许正是他们是追求优越感的体现，只不过，因为他们还小，迷了路，走错方向罢了。

试着判断孩子们的这些行为，背后的原因是什么：
● 那些阅读量远超其他同龄人的孩子，如何判断？
● 孩子表现出各种劣势，比如懒惰、装病、口吃等，他的心理是怎样的？（用劣势维持优势）
● 那些表现很自负的孩子？
● 那些喜欢吹牛的孩子？
● 那些绝口不提自己父母情况的孩子？
● 那些很容易被激怒的孩子？
● 那些具有攻击倾向的孩子？
● 那些喜欢贬低和折损别人的孩子？
● 那些不断突出自己，罔顾他人的孩子？
● 一个孩子自愿与世隔绝，说明什么？
● 那些有超出寻常的搜集物品癖好的孩子？
● 孩子很大了，突然又一天又开始尿床的孩子，如何解释他的行为？
● 我们把雄心视为优点，鼓励孩子多往这方面努力，这样真的对吗？
● 为什么一些小孩或者成年人只喜欢比他差的人交往？为什么有些人找对象喜欢找温柔听话的？
以上提到的现象，都可以尝试用“自卑情结”来解释。

反复提及的“自卑情结”

“自卑”二字贯穿全书。哪些孩子更容易产生自卑感呢？里面提到三类：父母过于严厉的、父母过于宠爱的、天生缺陷的孩子。这三类孩子的自卑都有其根源。

阿德勒年幼时应该有这种经历，否则怎能如此细致剖析自卑者各种卑鄙而细微的心理？阿德勒笔下的“自卑”并不是贬义词。他认为，自卑感和追求优越感同时存在，正是内心的自卑感刺激人改善自己的处境，追求优越，人们通过努力，消除自卑感获得成功（作者称之为“心理补偿”）。作为父母或者教育者要做的是引导他们走向正确的道路，消除自卑感。

既然“自卑”并不贬义，那我让它来得更猛烈一些，比如通过指责、责备等方式让孩子更自卑，这样不是更能刺激它消除自卑感吗？请放慢脚步，别忘了我们是中国人，最讲究“度”的一个群体。阿德勒指出，过度自卑感将给孩子带来膨胀的野心，同时会夸大遇到的困难，削弱孩子的勇气，逐渐他们演变为白日梦者，或者会走向另一条极端的道路，他们可能既羡慕又嫉妒比他们优秀的人，产生报复或恶毒心理，甚至犯罪等，以此获得短暂的成就，或者遇到失败后自暴自弃，恶性循环后走上不归路。观照一下自己对待孩子，有没有“太过”呢？

孩子成长中如何防止自卑

两个方面来看，一方面，我们分析孩子内心，如何引导他本自具足地防止发生自卑；另一方面，我们分析哪些因素会影响孩子，从而我们遏制这些因素，防止发生自卑。

1.如何引导孩子内心防止自卑
● 引导孩子不畏困难，即便失败也不放弃，将之看作是解决问题的机会，从而使孩子迅速强大起来。
● 注重鼓励而不仅仅是称赞（借用Dr.魏育儿宝典里面提过的观点），不要让孩子过分关注结果和评价。

2.如何遏制造成自卑的外部因素
● 孩子自身的缺陷
● 家中有人患病
● 孩子与死亡接触没有进行铺垫
● 父母很少花时间在孩子身上
● 家庭物质条件的变化
● 家中母亲强势控制一切
● 孩子在家中的位置（老大和老幺）
● 男性在女人堆长大，女性在男人堆长大
● 私生子的印记
● 换老师
● 留级
● 老师
● 陌生人的介入
● 与外祖父祖母的相处方式
● 家庭是否参与社会关系
● 接触的玩具（可见选择玩具要慎重）
● 与动物相处的方式
● 别人对孩子外表对评价
● 阅读书籍时，父母对书籍的解释

如何看待教育工作者

孩子会经历两个阶段的教育：父母家庭的教育、学校的教育。

1.第一阶段：父母家庭的教育

* 父母尤其是母亲应引导孩子把兴趣和信任扩展到其他人身上。孩子3岁之后，一定要走到社会关系中去。孩子的前三四年对于建立社会关系乃至于未来的社会关系认知都是这个年龄段建立的，过了这个时间，改善就费力多了。
* 不要单一鼓励孩子的雄心，雄心会让孩子更关注评价，更短视获得的成就，从而逃避和真正解决问题，造成人格障碍，走向一条错误的道路。
* 当孩子面对困难和失败的时候，我们要让孩子知道，面对失败不要放弃，要把遭遇失败当作新的问题解决。
* 应如何给孩子讲故事，不要让孩子主观臆断故事含义，作为大人，可以做出符合培养孩子良好人格的大方向的结论，减少孩子对故事和童话的误解。

2.第二阶段：学校的教育，第二阶段的教育恰好可以检验第一阶段教育有效与否。

* 哪些方面说明父母家庭阶段教育出现了问题？在学校的表现就可以看出来，尤其是初入学校的时候的表现。另外从其他一些迹象也可以看出来。例如催着起床，磨磨蹭蹭吃饭上学，丢失书本，逃学等。
* 怎样才是教育工作者的应有的态度？教育工作者对社会而不是对学校负责人，这样的绩效导向结果会引导孩子更有利于社会发展的方向走。那么，教育工作者就不会只追求表面的孩子好行为，而是能将孩子真正培养成一个人格健全的人。
* 面对表现没有远大志向的孩子，打低分数，严厉责罚不是正路，这种方式唤起雄心是一条不归路。应该以关切和温柔的方式对待他们，唤起他们的信心，引导他们找到更擅长的地方，补偿自卑感。
* 教育工作者应具有耐心。人格是一个统一体，是一环接着一环的因果联系，它是比较稳定的，孩子稍微大些后，期待一次谈话和鼓励就改变他是不可能的，所以，面对已经有问题的孩子，应该抱有耐心。
